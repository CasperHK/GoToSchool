SchoolSimple
============

## Introduction
A completely open-source web based school management tool using in Golang, ReactJS, SQLite and Webpack.

 * Go Language

 * ReactJS
   This is used to build the client side web application.

 * Webpack
   This is used to manage Javascript.

 * SQLite
